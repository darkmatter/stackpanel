# stackpanel caddy add

Add a reverse proxy site to Caddy.

Examples:
  stackpanel caddy add myapp.localhost localhost:3000
  stackpanel caddy add api.localhost localhost:8080 --tls

## Usage

```bash
stackpanel caddy add [domain] [upstream]
```


## Flags

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--tls` | `bool` | `false` | Enable internal TLS |

# stackpanel caddy add

Add a reverse proxy site to Caddy.

Examples:
  stackpanel caddy add myapp.localhost localhost:3000
  stackpanel caddy add api.localhost localhost:8080 --tls

## Usage

```bash
stackpanel caddy add [domain] [upstream]
```


## Flags

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--tls` | `bool` | `false` | Enable Caddy's built-in self-signed TLS |
| `--tls-cert` | `string` | _none_ | Path to TLS certificate file (e.g., from step ca certificate) |
| `--tls-key` | `string` | _none_ | Path to TLS private key file |
| `--tls-step-ca-root` | `string` | _none_ | Path to Step CA root certificate |
| `--tls-step-ca-url` | `string` | _none_ | Step CA ACME directory URL for TLS certs |

# stackpanel vars list

List all variables defined in stackpanel.variables.

Variables are grouped by their path prefix (e.g., /apps, /services, /dev, /prod).
Use --type to filter by variable type.

## Usage

```bash
stackpanel vars list
```


## Flags

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--json` | `bool` | `false` | Output as JSON |
| `--type, -t` | `string` | _none_ | Filter by type (LITERAL, SECRET, VALS, EXEC) |

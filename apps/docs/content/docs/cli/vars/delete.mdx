# stackpanel vars delete

Remove a variable from the workspace.

Examples:
  stackpanel vars delete /my/api-host
  stackpanel vars rm my/db-port

## Usage

```bash
stackpanel vars delete <variable-id>
```

**Aliases:** `rm`, `remove`

---
title: Stackpanel Secrets
description: Documentation for the secrets module
---

Stackpanel secrets now rely on Nix-configured SOPS recipients.

- Groups are file buckets such as `vars/dev.sops.yaml`.
- Recipient public keys live in `stackpanel.secrets.recipients` (or fall back from `stackpanel.users`).
- `.stack/secrets/.sops.yaml` is generated from those recipients.
- There is no extra per-group private key layer in the secrets flow.

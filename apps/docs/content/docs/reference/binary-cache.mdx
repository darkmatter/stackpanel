# Binary-cache Options

## `binary-cache.cachix.cache`

Cachix cache name for pushing binaries

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `""` |

---

## `binary-cache.cachix.enable`

Whether to enable Cachix integration.

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `false` |

**Example:**

`true`


---

## `binary-cache.cachix.token-path`

Path to a Cachix auth token file

| Property | Value |
|----------|-------|
| **Type** | `null or absolute path` |
| **Default** | `null` |

---

## `binary-cache.enable`

Whether to enable Binary cache configuration.

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `false` |

**Example:**

`true`

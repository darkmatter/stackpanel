# Commands Options

## `commands`

# Stackpanel commands

| Property | Value |
|----------|-------|
| **Type** | `attribute set of (submodule)` |
| **Default** | `{ }` |

**Example:**

```nix
{
  "db:seed": {
    package = "npm";
    command = "run seed";
    domain = "localhost";
    env = {
      NODE_ENV = "development";
    };
    cwd = null;
  }
}
```

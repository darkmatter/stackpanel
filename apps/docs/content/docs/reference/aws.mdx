# Aws Options

## `stackpanel.aws.certAuth.accountId`

AWS account ID

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | _none_ |


---

## `stackpanel.aws.certAuth.cacheBufferSeconds`

Seconds before expiry to refresh cached credentials

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `"300"` |


---

## `stackpanel.aws.certAuth.enable`

Whether to enable AWS Roles Anywhere cert auth.

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `false` |


**Example:**

`true`

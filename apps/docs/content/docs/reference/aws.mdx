# Aws Options

## `aws.certAuth.account-id`

AWS account ID

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `""` |

---

## `aws.certAuth.cache-buffer-seconds`

Seconds before expiry to refresh cached credentials

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `"300"` |

---

## `aws.certAuth.enable`

Whether to enable AWS Roles Anywhere cert auth.

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `false` |

**Example:**

`true`

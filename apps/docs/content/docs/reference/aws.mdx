# Aws Options

## `aws.default-profile`

AWS profile name to use as default. Empty string means 'default'.

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `""` |

---

## `aws.extra-config`

Additional AWS config to append (raw INI format).
Use this to define extra profiles or settings.

Example:
  extra-config = ''
    [profile production]
    region = us-east-1
    role_arn = arn:aws:iam::123456789012:role/ProdRole
    source_profile = default
  '';

| Property | Value |
|----------|-------|
| **Type** | `strings concatenated with "\n"` |
| **Default** | `""` |

---

## Quick Start

```nix
# In your devenv.nix
{
  stackpanel = {
    enable = true;

    # Port management
    ports.projectName = "myproject";

    # Services
    globalServices.postgres.enable = true;
    globalServices.redis.enable = true;

    # Theme
    theme.enable = true;
  };
}
```

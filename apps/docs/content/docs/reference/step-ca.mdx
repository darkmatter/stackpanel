# Step-ca Options

## `step-ca.ca-fingerprint`

Step CA root certificate fingerprint

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `""` |

---

## `step-ca.ca-url`

Step CA URL (e.g., https://ca.internal:443)

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `""` |

---

## `step-ca.cert-name`

Common name for the device certificate

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `"device"` |

---

## `step-ca.enable`

Whether to enable Step CA certificate management.

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `false` |

**Example:**

`true`


---

## `step-ca.prompt-on-shell`

Prompt for certificate setup on shell entry if not configured

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `true` |

---

## `step-ca.provisioner`

Step CA provisioner name

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `"Authentik"` |

---

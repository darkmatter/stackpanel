# State Options

## `state.enable`

Whether to enable state file generation.

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `true` |

**Example:**

`true`


---

## `state.file`

Name of the state file in stateDir

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `"stackpanel.json"` |

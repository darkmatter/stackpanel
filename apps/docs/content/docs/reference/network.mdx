# Network Options

## `network.step.ca-fingerprint`

Step CA root certificate fingerprint

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `""` |

---

## `network.step.ca-url`

Step CA URL (e.g., https://ca.internal:443)

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `""` |

---

## `network.step.cert-name`

Common name for the device certificate

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `"device"` |

---

## `network.step.enable`

Whether to enable Step CA certificate management.

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `false` |

**Example:**

`true`


---

## `network.step.enterShell`

Scripts to run when entering the shell (internal)

| Property | Value |
|----------|-------|
| **Type** | `strings concatenated with "\n"` |
| **Default** | `""` |

---

## `network.step.prompt-on-shell`

Prompt for certificate setup on shell entry if not configured

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `true` |

---

## `network.step.provisioner`

Step CA provisioner name

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `"Authentik"` |

---

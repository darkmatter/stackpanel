# Direnv Options

## `stackpanel.direnv.hideEnvDiff`

Hide the 'direnv: export +VAR...' log line (requires user's direnv.toml)

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `true` |

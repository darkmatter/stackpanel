# Theme Options

## `stackpanel.theme.configFile`

Custom starship.toml config file (uses stackpanel default if not set)

| Property | Value |
|----------|-------|
| **Type** | `null or absolute path` |
| **Default** | `null` |


---

## `stackpanel.theme.enable`

Whether to enable Starship prompt for stackpanel devenv.

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `false` |


**Example:**

`true`

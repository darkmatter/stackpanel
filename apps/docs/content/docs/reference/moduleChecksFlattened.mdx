# ModuleChecksFlattened Options

## `moduleChecksFlattened`

Flattened module check derivations for use in flake checks output.
Keys are "-" (e.g., "oxlint-eval").

| Property | Value |
|----------|-------|
| **Type** | `attribute set of package` |
| **Default** | `{ }` |
| **Read Only** | `true` |

# Gitignore Options

`stackpanel.gitignore` is a reserved core option for managing a block-owned section in the project root `.gitignore`.

The managed block is:

- deduplicated
- sorted
- merged from core defaults + your custom entries
- isolated so user-written content outside the block is preserved

---

## `gitignore.enable`

Whether Stackpanel should manage a `.gitignore` block.

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `true` |

---

## `gitignore.entries`

Additional `.gitignore` entries to include in the managed block.

| Property | Value |
|----------|-------|
| **Type** | `list of string` |
| **Default** | `[]` |

Example:

```nix
stackpanel.gitignore.entries = [
  "dist/"
  ".env.local"
  "coverage/"
];
```

---

## `gitignore.defaults.stackpanelState`

Include Stackpanel runtime state directory in `.gitignore`:

- `${stackpanel.dirs.home}/state/`

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `true` |

---

## `gitignore.defaults.localConfig`

Include local override config in `.gitignore`:

- `${stackpanel.dirs.home}/config.local.nix`

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `true` |

---

## `gitignore.defaults.tasksDir`

Include generated task symlink directory in `.gitignore`:

- `.tasks`

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `true` |

---

## `gitignore.defaults.projectMarker`

Include the Stackpanel root marker file in `.gitignore`:

- `stackpanel.root-marker` (default: `.stackpanel-root`)

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `false` |

---

## `gitignore.defaults.addProjectMarker` (deprecated)

Deprecated alias for `gitignore.defaults.projectMarker`.

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `false` |

Use `gitignore.defaults.projectMarker` for new config.

---

## Example

```nix
stackpanel.gitignore = {
  enable = true;

  defaults = {
    stackpanelState = true;
    localConfig = true;
    tasksDir = true;
    projectMarker = true;
  };

  entries = [
    "dist/"
    ".env.local"
    "coverage/"
  ];
};
```

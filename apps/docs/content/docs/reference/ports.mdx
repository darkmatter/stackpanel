# Ports Options

## `stackpanel.ports.basePort`

The computed base port for this project

| Property | Value |
|----------|-------|
| **Type** | `16 bit unsigned integer; between 0 and 65535 (both inclusive)` |
| **Default** | `6400` |
| **Read Only** | `true` |


---

## `stackpanel.ports.enable`

Whether to enable Automatic port assignment.

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `true` |


**Example:**

`true`


---

## `stackpanel.ports.minPort`

Minimum port number for the port range

| Property | Value |
|----------|-------|
| **Type** | `16 bit unsigned integer; between 0 and 65535 (both inclusive)` |
| **Default** | `3000` |


---

## `stackpanel.ports.modulus`

Port rounding modulus. Ports are rounded to nearest multiple.
Use 100 for ports like 3100, 3200, 3300.
Use 10 for ports like 3110, 3120, 3130 (if collision occurs).

| Property | Value |
|----------|-------|
| **Type** | `signed integer` |
| **Default** | `100` |


**Example:**

`10`


---

## `stackpanel.ports.portRange`

Range of ports to use (minPort to minPort + portRange)

| Property | Value |
|----------|-------|
| **Type** | `signed integer` |
| **Default** | `7000` |


---

## `stackpanel.ports.projectName`

Project name used for port computation. Set this to match your project name.

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `"default"` |


**Example:**

`"myapp"`


---

## `stackpanel.ports.service`

Computed service information by key.
Access: config.stackpanel.ports.service.POSTGRES.port

| Property | Value |
|----------|-------|
| **Type** | `attribute set of attribute set of unspecified value` |
| **Default** | _see below_ |
| **Read Only** | `true` |


**Default:**

```nix
{
  MINIO = {
    displayName = "Minio";
    key = "MINIO";
    name = "Minio";
    port = 6412;
  };
  MINIO_CONSOLE = {
    displayName = "Minio Console";
    key = "MINIO_CONSOLE";
    name = "Minio Console";
    port = 6413;
  };
  POSTGRES = {
    displayName = "PostgreSQL";
    key = "POSTGRES";
    name = "PostgreSQL";
    port = 6410;
  };
  REDIS = {
    displayName = "Redis";
    key = "REDIS";
    name = "Redis";
    port = 6411;
  };
}
```


---

## `stackpanel.ports.services`

List of infrastructure services that need ports.
Each service gets a port at basePort + 10 + index.
Environment variable: STACKPANEL__PORT

| Property | Value |
|----------|-------|
| **Type** | `list of (submodule)` |
| **Default** | `[ ]` |


**Example:**

```nix
[
  { key = "POSTGRES"; name = "PostgreSQL"; }
  { key = "REDIS"; name = "Redis"; }
  { key = "MINIO"; name = "Minio"; }
  { key = "MINIO_CONSOLE"; name = "Minio Console"; }
]
```


---

## `stackpanel.ports.services.*.key`

Unique key for the service (used in env var: STACKPANEL__PORT)

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | _none_ |


**Example:**

`"POSTGRES"`


---

## `stackpanel.ports.services.*.name`

Human-readable name for display

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `""` |


**Example:**

`"PostgreSQL"`

# Ports Options

<NixOption
  path={"ports.base-port"}
  type={"16 bit unsigned integer; between 0 and 65535 (both inclusive)"}
  defaultValue={"3600"}
  readonly
  module={"ports"}
>
The computed base port for this project
</NixOption>

<NixOption
  path={"ports.enable"}
  type={"boolean"}
  defaultValue={"true"}
  exampleCode={"true"}
  module={"ports"}
>
Whether to enable Automatic port assignment.
</NixOption>

<NixOption
  path={"ports.project-name"}
  type={"string"}
  defaultValue={"\"default\""}
  exampleCode={"\"myapp\""}
  module={"ports"}
>
Project name used for port computation. Set this to match your project name.
</NixOption>

<NixOption
  path={"ports.service"}
  type={"attribute set of attribute set of unspecified value"}
  defaultValue={"{ }"}
  readonly
  module={"ports"}
>
Computed service information by key.
Access: config.stackpanel.ports.service.POSTGRES.port
</NixOption>

<NixOption
  path={"ports.services"}
  type={"attribute set of (submodule)"}
  defaultValue={"{ }"}
  exampleCode={"{\n  POSTGRES = { name = \"PostgreSQL\"; };\n  REDIS = { name = \"Redis\"; };\n  MINIO = { name = \"Minio\"; };\n  MINIO_CONSOLE = { name = \"Minio Console\"; };\n}"}
  module={"ports"}
>
Attrset of infrastructure services that need ports.
Each service gets a deterministic port based on project name + service key.
Provided via STACKPANEL_SERVICES_CONFIG (JSON).
</NixOption>

<NixOption
  path={"ports.services.<name>.name"}
  type={"string"}
  defaultValue={"\"‹name›\""}
  exampleCode={"\"PostgreSQL\""}
  module={"ports"}
>
Human-readable name for display (defaults to attrset key)
</NixOption>

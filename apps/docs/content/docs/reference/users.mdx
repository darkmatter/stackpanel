# Users Options

## `users`

Users of the repository who should have access to secrets.

Base schema: nix/stackpanel/db/schemas/users.proto.nix
Extended with: is-admin, email

| Property | Value |
|----------|-------|
| **Type** | `attribute set of (submodule)` |
| **Default** | `{ }` |

**Example:**

```nix
{
  alice = {
    github = "aliceexample";
    name = "Alice Example";
    public-keys = [
      "age1..."
    ];
    secrets-allowed-environments = [
      "dev"
    ];
  };
  cooper = {
    email = "cooper@example.com";
    github = "coopermaruyama";
    is-admin = true;
    name = "Cooper Maruyama";
    public-keys = [
      "age1..."
      "ssh-ed25519 AAAA..."
    ];
    secrets-allowed-environments = [
      "dev"
      "staging"
      "production"
    ];
  };
}
```

## `users.<name>.email`

Email address for notifications

| Property | Value |
|----------|-------|
| **Type** | `null or string` |
| **Default** | `null` |

---

## `users.<name>.github`

GitHub username

| Property | Value |
|----------|-------|
| **Type** | `null or string` |
| **Default** | `null` |

---

## `users.<name>.is-admin`

Whether this user has admin privileges (computed at runtime)

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `false` |

---

## `users.<name>.name`

Display name of the user

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `""` |

---

## `users.<name>.public-keys`

SSH or AGE public keys for encryption

| Property | Value |
|----------|-------|
| **Type** | `list of string` |
| **Default** | `[ ]` |

---

## `users.<name>.secrets-allowed-environments`

Environments this user can access secrets for

| Property | Value |
|----------|-------|
| **Type** | `list of (attribute set)` |
| **Default** | `[ ]` |

# GlobalServices Options

## `stackpanel.globalServices.caddy.enable`

Whether to enable Global Caddy reverse proxy.

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `false` |


**Example:**

`true`


---

## `stackpanel.globalServices.caddy.sites`

Sites to register with Caddy (domain -> upstream)

| Property | Value |
|----------|-------|
| **Type** | `attribute set of string` |
| **Default** | `{ }` |


**Example:**

```nix
{
  "api.localhost" = "localhost:8080";
  "myapp.localhost" = "localhost:3000";
}
```


---

---

## `stackpanel.globalServices.minio.consolePort`

Minio console port. If null, uses computed port from stackpanel.ports.

| Property | Value |
|----------|-------|
| **Type** | `null or 16 bit unsigned integer; between 0 and 65535 (both inclusive)` |
| **Default** | `null` |

## `stackpanel.globalServices.postgres.databases`

List of databases to create for this project

| Property | Value |
|----------|-------|
| **Type** | `list of string` |
| **Default** | _see below_ |


**Default:**

```nix
[
  "stackpanel"
]
```


**Example:**

```nix
[
  "myapp"
  "myapp_test"
]
```

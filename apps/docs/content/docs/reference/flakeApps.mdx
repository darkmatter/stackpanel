# FlakeApps Options

## `flakeApps`

Flake apps to expose via `nix run .#`.

Each app must have:
  - type: "app"
  - program: Path to executable (usually from a derivation)

Example:
  stackpanel.flakeApps = \{
    web = \{ type = "app"; program = "$\{myPackage\}/bin/web"; \};
  \};

| Property | Value |
|----------|-------|
| **Type** | `attribute set of (submodule)` |
| **Default** | `{ }` |

---

## `flakeApps.<name>.program`

Path to the executable.

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | _none_ |

---

## `flakeApps.<name>.type`

App type (always 'app' for Nix apps).

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `"app"` |

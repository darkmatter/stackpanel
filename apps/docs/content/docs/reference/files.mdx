# Files Options

## `files.enable`

Whether to enable Generate arbitrary files into the repo.

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `false` |

**Example:**

`true`


---

## `files.exeFilename`

Name of the generated writer executable.

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `"write-files"` |

---

## `files.files`

List of files to write (path relative to repo root) + derivation with contents.

| Property | Value |
|----------|-------|
| **Type** | `list of (submodule)` |
| **Default** | `[ ]` |

---

## `files.files.*.drv`

Derivation whose outPath is a file containing the desired content.

| Property | Value |
|----------|-------|
| **Type** | `package` |
| **Default** | _none_ |

---

## `files.files.*.mode`

Optional chmod mode (e.g. 0644, 0755).

| Property | Value |
|----------|-------|
| **Type** | `null or string` |
| **Default** | `null` |

---

## `files.files.*.path`

File path relative to repo root.

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | _none_ |

**Example:**

`".github/workflows/ci.yml"`


---

## `files.rootVar`

Environment variable that points at the repo root.

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `"STACKPANEL_ROOT"` |

# Cli Options

## `cli.enable`

Whether to enable CLI-based file generation.

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `false` |

**Example:**

`true`


---

## `cli.quiet`

Suppress generation output messages

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `false` |

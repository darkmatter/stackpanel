# Secrets Options

## `secrets.codegen`

Code generation settings for secrets.

| Property | Value |
|----------|-------|
| **Type** | `attribute set of (submodule)` |
| **Default** | `{ }` |

---

## `secrets.codegen.<name>.directory`

Output directory for generated code.

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | _none_ |

---

## `secrets.codegen.<name>.language`

Programming language for generated code.

| Property | Value |
|----------|-------|
| **Type** | `one of "typescript", "go"` |
| **Default** | _none_ |

---

## `secrets.codegen.<name>.name`

Name of the generated code package.

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | _none_ |

---

## `secrets.enable`

Enable StackPanel secrets utilities.

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `false` |

---

## `secrets.environments`

Environment-specific secrets configurations.

| Property | Value |
|----------|-------|
| **Type** | `attribute set of (submodule)` |
| **Default** | `{ }` |

---

## `secrets.environments.<name>.name`

Name of the environment.

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | _none_ |

---

## `secrets.environments.<name>.public-keys`

List of AGE public keys that can decrypt this environment.

| Property | Value |
|----------|-------|
| **Type** | `list of string` |
| **Default** | `[ ]` |

---

## `secrets.environments.<name>.sources`

List of SOPS-encrypted source files for this environment (without .yaml extension).

| Property | Value |
|----------|-------|
| **Type** | `list of string` |
| **Default** | `[ ]` |

---

## `secrets.input-directory`

Directory where your secrets are stored - should contain a SOPS-encrypted file per environment.

| Property | Value |
|----------|-------|
| **Type** | `string` |
| **Default** | `".stackpanel/secrets"` |

---

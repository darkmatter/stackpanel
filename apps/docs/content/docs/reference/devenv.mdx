# Devenv Options

## `stackpanel.devenv.recommended.enable`

When enabled, applies recommended settings for stackpanel.devenv integration.

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `false` |


**Example:**

`true`


---

## `stackpanel.devenv.recommended.formatters.enable`

When enabled, applies recommended settings for code formatters in the dev environment.

| Property | Value |
|----------|-------|
| **Type** | `boolean` |
| **Default** | `false` |


**Example:**

`true`

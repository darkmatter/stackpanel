---
title: "CLI-based file generation for stackpanel"
description: This module aggregates configuration and invokes `stackpanel init` CLI
---

This module aggregates configuration and invokes `stackpanel init` CLI
when entering the devshell. All codegen is kicked off as a result.

Generated files (by CLI):
```nix
- .stackpanel/state/stackpanel.json (runtime state)
- .stackpanel/gen/ide/vscode/* (VS Code workspace, loader script)
- .stackpanel/gen/schemas/secrets/* (JSON schemas from Nix definitions)
```

Go application support using gomod2nix for hermetic packaging.

## Components

- meta.nix: Static metadata for discovery
- module.nix: Options, packages, scripts, health checks
- ui.nix: UI panel definitions

## Usage

```nix
stackpanel.apps.my-app = {
  path = "apps/my-app";
  go.enable = true;
};
```
